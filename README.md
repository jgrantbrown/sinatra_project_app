# sinatra_project_app

I was there.

Track the concerts you have gone to. See bands that have played at a certain Venue. Check how many times your friend has been to a Venue or seen a Band.

Usage

To use this app, just clone, run rake db:migrate and then run shotgun. Everything should be set up.
